# 複数図形の描画

## 要求事項

- 矩形 (Rectangle) をマウスドラッグで描画できる
- 円 (Circle) をマウスドラッグで描画できる
- 楕円 (Ellipse) をマウスドラッグで描画できる
- 複合線 (Polyline) をクリックで頂点を追加しながら描画できる
- 多角形 (Polygon) をクリックで頂点を追加しながら描画できる (最後に始点へ閉じる)
- 描いた図形はウィンドウを再描画しても消えない
- 各図形をツールバーで切り替えて描画できる
- 図形には塗り (fill) を設定できる

## 達成基準

`W3C_SVG_11_TestSuite/png/shapes-*.png` に含まれる全図形種別
(circle，ellipse，line，polygon，polyline，rect) をエディタ上で描画できること．
ただし，本ワークパッケージで実装する機能はベクトルデータの作成と塗りに限定し，
線幅・破線・その他の細かいスタイル設定は対象外とする．

達成確認は，JUnit テスト内で各図形を Java コードで生成・描画し，参照 PNG と
並べて表示したうえで `JOptionPane` による目視確認ダイアログを通じて行う．
確認ダイアログで「Yes」を選択した場合にテスト合格とする．

## 方針

static-structure.md のクラス構成を基本とし，前ワークパッケージで実装済みの Line
と同様の方式で各図形クラスとツールクラスを追加する．

- `model` パッケージに `Rectangle`，`Circle`，`Ellipse`，`Polyline`，`Polygon`
  を実装 (`Figure` を継承)
- `controller` パッケージに `DrawRectTool`，`DrawCircleTool`，`DrawEllipseTool`，
  `DrawPolylineTool`，`DrawPolygonTool` を実装 (`Tool` インタフェースを実装)
- `Polyline` / `Polygon` は頂点リストを保持し，クリックごとに頂点を追加，
  ダブルクリックまたは始点クリックで確定する
- `EditorFrame` にツール切替ボタンを追加し，`EditorController.setTool` を呼び出す
- `Figure` 基底クラスに `fillColor` フィールドを追加し，各図形の `draw` で塗りを適用する
- 線幅・破線など細かいスタイル設定は本ワークパッケージでは実装しない
- 各図形クラスのユニットテストを作成
- 各図形種別について，Java コードで Figure を生成して Canvas に描画し，参照 PNG と
  並べて表示する目視確認テストを作成する (JUnit + JOptionPane による Yes/No 確認)

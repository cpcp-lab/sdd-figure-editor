# 要求仕様

## 目的

Java および Swing を用いたオブジェクト指向設計の学習を目的として，図形エディタを実装する．
図形の描画・選択・移動・削除などの基本操作を通じて，継承・ポリモーフィズム・デザインパターンの適用を実践する．

## おもな機能

### 図形の描画

- キャンバス上でマウス操作により図形を新規作成できる
- 作成できる図形の種類: 線分，矩形， (将来的に円なども対象)
- 複数の図形をグループ化し，一つの図形として扱える

### 図形の選択・操作

- キャンバス上の図形をマウスクリックで選択できる
- 選択した図形を移動できる
- 選択した図形を削除できる

### 表示

- キャンバスに描画された図形を常時表示する
- 選択状態の図形は視覚的に区別して表示する

### SVG ファイル入出力

- キャンバス上の図形を SVG ファイルとして保存できる
- 保存した SVG ファイルをブラウザや SVG ビューアで正しく表示できる
- SVG ファイルを読み込み，キャンバスに図形を再現できる
- サポートする SVG 要素: `<line>`, `<rect>` (角丸対応)，`<circle>`, `<ellipse>`, `<polyline>`, `<polygon>`
- サポートする SVG 属性: `fill`, `stroke`, `stroke-width`，グループ要素 `<g>` によるスタイル継承と `translate` 変換
- メニューバーの File > Save / Open からファイル操作を行う

## 将来的な拡張 (予定外)

- undo/redo 操作
- 図形の色・線幅の変更
